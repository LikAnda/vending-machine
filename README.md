# Vending Machine (raspi)
A small things vending machine with a raspberry.

----

## Installation

- `sudo npm install`

- Motion sensor on pin 11 (GPIO 17)

- Piezo connected to pin 13 (GPIO 27)

----

## Run

- `sudo node index.js`

- To activate the system remotely, go to `ip-of-the-raspberry:2000`

----

### Packages

- express

- johny-five

- raspi-io

- socket.io
